# HA Solar Excess Optimizer

Home Assistant Add-on Repository für die modulare PV-Überschussregelung.

## Installation

1. [![Add Repository](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https://github.com/DEIN-USERNAME/ha-solar-excess-optimizer)

   **Oder manuell:**
   Einstellungen → Add-ons → Add-on Store → ⋮ → Repositories → diese URL einfügen

2. „HA Solar Excess Optimizer" im Store suchen & installieren

## Add-ons in diesem Repository

### ☀️ HA Solar Excess Optimizer
Modulare PV-Überschussregelung mit 4 Gerätetypen:
- **Switch** – einfaches An/Aus
- **Stepped** – mehrere fixe Leistungsstufen (z.B. Heizstab 1/2/3 kW)
- **Variable** – stufenlose Regelung (z.B. Wallbox via Ampere)
- **Timed** – Mindestlaufzeit pro Tag
